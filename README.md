**📊 RFM Customer Segmentation Analysis**


**🚀 Project Overview**

This project uses RFM (Recency, Frequency, Monetary) analysis to segment customers based on purchasing behavior and customer value.

The objective is to identify high-value customers, at-risk customers, and customer engagement patterns to support data-driven marketing and retention strategies.


**🎯 Business Problem**
Businesses often struggle to identify:

Which customers generate the most value
Which customers are becoming inactive
Which customer segments require retention campaigns
How purchasing behavior impacts customer lifetime value

This project helps organizations improve customer retention, personalization, and marketing effectiveness through customer segmentation.
